Portfólio profissional da Designer Gráfica **Giovana Kymberli**, desenvolvido para apresentar projetos criativos, identidades visuais, branding e campanhas publicitárias de forma moderna, organizada e responsiva.

## ✨ Sobre o Projeto

Este portfólio foi criado com o objetivo de reunir e exibir os principais trabalhos desenvolvidos por Giovana Kymberli, proporcionando uma navegação intuitiva e uma apresentação visual profissional dos projetos.

O site conta com:

- Apresentação profissional da designer
- Galeria de projetos organizada por categorias
- Visualização ampliada das imagens (Lightbox)
- Layout responsivo para desktop e dispositivos móveis
- Interface moderna e focada na experiência do usuário

## 📂 Categorias de Projetos

- 📱 Social Media
- 🎨 Branding de Marca
- 📖 Projeto Editorial (Revista)
- 📢 Campanhas Publicitárias

## 📸 Funcionalidades

- Navegação entre projetos
- Galerias de imagens
- Lightbox para visualização ampliada
- Layout responsivo
- Interface otimizada para portfólio profissional
